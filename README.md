# Word-Game
Welcome to my Python lab 12 project, consisting of a fun, customizable word game, playable with friends!

# Description
A fun, nostalgic game for friends and family, or even by yourself! The goal of the game is to guess what word is given to you in 10 tries or less or else you lose! With other fun customizable options, such as adding or subtracting from the list of possible words, or viewing what may be in your current word bank, the power of this word game is in your hands! I do advise you to use the add or view list functions only if you're playing with friends, as it'll take the fun out of the game for you.

# Installation/Usage
As soon as you download the folder with both files, whether you're in VS Code or your terminal, as long as you're in the right file directory, you can run the program as normal, and it should start right up!

Terminal command: python3 main.py
